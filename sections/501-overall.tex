\subsection{Overall Progress}
% RQ1: agent能进展多远——max-based平均50.1，glm-5.2最强，P5/R6几乎不可达
Table~\ref{tab:model-results} reports the best-of-three results for each model across 30 CVEs. \texttt{glm-5.2} achieves the highest average overall score (65.2/100), followed by \texttt{mimo-v2.5-free} (50.3) and \texttt{claude-sonnet-4-6} (49.9). While real-target success remains rare, only 17 out of 150 CVE--model pairs achieve near-full P5 and P6 credit, the best single run reaches 99.5/100, demonstrating that full autonomous reproduction is achievable under favorable conditions.

Specifically, there are four CVEs that achieve full or near-full reproduction, which share
a common reason: the target devices use ARM64 processors, allowing agents to run the
real firmware binary via \textit{chroot} without cross-architecture emulation. In contrast, MIPS-based devices requiring QEMU emulation almost universally fail at P5, because agents cannot reliably configure NVRAM, shared libraries, and network bridges.


Notably, the leading model, \texttt{glm-5.2}, wins not through raw reasoning but through task-appropriate behavior. It averages 7.7/15 on P2 (Firmware Acquisition), nearly double \texttt{claude-sonnet-4-6}'s 4.5/15; since P2 gates all downstream phases, this gap cascades---\texttt{claude-sonnet-4-6} has 17 runs where the plan exceeds 50 yet the task stalls below 15, all blocked at P2. \texttt{claude-sonnet-4-6} also substitutes via simulation at the highest rate (81.1\% of runs vs.\ \texttt{glm-5.2}'s 56.7\%), while \texttt{glm-5.2} achieves the strongest fallback planning (37.2/100 vs.\ 25.8--31.7 for the others), which correlates with deeper pipeline progression. On from-scratch reproduction, persistence in artifact acquisition and fallback-driven recovery matter more than model size.
